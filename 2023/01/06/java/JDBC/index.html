# JDBC

# 介绍

JDBC: Java Database Connectivity | Java 连接数据库技术

使用 JDBC 提供的方法，可以发送字符串类型的 SQL 语句到数据库管理软件，并且获取语句执行结果，进而实现数据库数据 CURD 操作的技术

  • JDBC 是 Java 连接数据库技术的统称

  • JDBC 由两部分组成：

    1. Java 提供的 JDBC 规范（接口）

    2. 各个数据库厂商的实现驱动 jar 包

  • JDBC 技术是一种典型的面向接口编程

# 使用步骤

  1. 注册驱动
  2. 获取连接
  3. 创建发送 sql 语句对象
  4. 发送 sql 语句，并获取返回结果
  5. 结果解析
  6. 资源释放

# 存在问题

  1. SQL 语句需要字符串拼接，比较麻烦
  2. 只能拼接字符串类型，其他数据库类型无法处理
  3. 可能发送注入攻击

# 主键回显和主键值获取

  1. 创建 prepareStatement 的时候，告知，携带数据库自增长的主键 (sql, Statement.RETURN_GENERATED_KEYS)
  2. 获取装主键值结果集对象，一行一列获取对应数据

# 批量插入

  1. 路径后面添加？rewriteBatchedStatements=true 允许批量插入
  2. INSERT INTO … VALUES [必须写]
  3. 不执行每条语句，批量添加 addBatch ()
  4. 遍历添加完毕，统一批量执行 executeBatch ()

# 转帐事务的实现

  1. 事务添加是在业务方法中
  2. 利用 try…catch 代码块，开启事务和提交事务，和事务回滚
  3. 将 connection 传入 Dao，只负责使用不用关闭
